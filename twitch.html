Overview

This app was built using React JS and is hosted on Firebase. Users area able to search properties based on an Address, City, or Zip code to retrieve a list of active properties currently for sale. You will be able to view property information as well as the specific location of the property integrated with Google Maps API. User authentication is availabe so you can signup & signin to your account with an email address in order to save your favorite properties. This is made possible with Zillow API.
